initialValue

initialValue() 方法是 ThreadLocal 中一个能被重写的方法，作用是提供初始值。

initialValue() 方法会在首次调用 get() 方法时被调用，但是，有几个例外：

  1. 在首次调用 get() 方法之前若调用过 set() 方法，则不会被调用。
  2. 若在调用 remove() 方法之后紧接着调用 get() 方法，则会被再次调用。

总而言之，就是在 thread local variable 无值时提供初始值。
