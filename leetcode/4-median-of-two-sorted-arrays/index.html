查找中位数是查找第 k 小的特殊情况。若只有一个数组，设其长度为 n：

  • 若 n 是奇数，则中位数是 nums[k], k = n // 2。
  • 若 n 是偶数，则中位数是 (nums[k-1] + nums[k]) / 2, k = n // 2。

※ 从定义出发的二分查找

中位数将集合分成两个大小相等的部分，使得左半部分中的任意元素都小于等于右半部分的任意元素。

我们可以对数组进行切割，当长度为奇数时，使左半部分的大小比右半部分多一个元素：

当有两个数组时，设第一个数组的分割点在 i, 0 <= i <= n，第二个数组的分割点在 j, 0 <= j <= m，需要保证 i + j = n - i + m - j（总长度为偶数情况），或者 i + j = n - i + m - j + 1（总长度为奇数情况），二者结合，j = (n + m + 1) // 2 - i。通过整除消除了奇偶之间的差异。

移动分割点，使之满足”左半部分中的任意元素都小于等于右半部分的任意元素“这一条件，就能得到中位数。只需要移动其中一个分割点 i，另一个分割点 j 的位置可通过计算得到。怎么移动？二分查找法。

初始时，i 取 [0, n] 的中间位置，计算出 j，由于 nums1 和 nums2 是已排序的，i 的移动方向取决于 nums1[i - 1] ? nums2[j] 和 nums2[j - 1] ? nums1[i] 的大小关系：

  • 若 nums1[i - 1] > nums2[j]，i 应该向左移，将较大的 nums1[i - 1] 分到右半部分去。
  • 反之，若 nums2[j - 1] > nums1[i]，i 应该向右移，将较小的 nums1[i] 分到左半部分去。

二者均不满足，说明 i 来到了正确的分割点，划分终止。
