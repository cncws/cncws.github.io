最大矩形

提示：

  • rows == matrix.length
  • cols == matrix[r].length
  • 0 <= row, cols <= 200
  • matrix[i][j] 为 '0' 或 '1'

※ 动态规划

每矩阵中的每一点，维护三个变量：

  • height[i][j]

    (i, j) 为底时的最大高度，即 (i, j) 上方有多少个连续的 '1'。

  • left[i][j]

    以 height[i][j] 为矩形的高时，矩形的左边界，即要求对所有的列 left[i][j] <= col < i，满足条件 height[i][col] >= height[i][j]。

  • right[i][j]

    以 height[i][j] 为矩形的高时，矩形的右边界。

有了这三个值，以 (i, j) 所在行为底边矩形面积等于 (right[i][j] - left[i][j] + 1) * height[i][j]。

如何更新这三个值是动态规划的重点：

  • height[i][j]：

    • 若 matrix[i][j] == '0'，则 height[i][j] = 0。
    • 若 matrix[i][j] == '1'，则可以和同列上一行的点连起来：height[i][j] = height[i - 1][j] + 1。

  • left[i][j]

    • 若 matrix[i][j] == '0'，则 left[i][j] = 0。

    • 若 matrix[i][j] == '1'，则还要观察同行之前所有列的值：只能扩展到最近的值为 '0' 的列。

  • right[i][j] 更新方式与 left[i][j] 类似

单调栈

将矩阵的一列看作一根柱子，则柱子的高度是动态规划中的 height[i][j]。

之后，求矩阵中的最大矩形就是求柱状图中最大的矩形，这里直接使用它的单调栈解法。
