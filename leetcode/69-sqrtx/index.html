x 的平方根

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

二分查找

当 left <= right 时，令 mid = (left + right) // 2：

  • 若 mid * mid <= x，令 left = mid + 1。
  • 若 mid * mid > x，令 right = mid + 1。

当 left > right 时，left - 1 就是问题的解。

※ 牛顿迭代法

求 \(\sqrt{C}\) 就是求 \(y = x^2 - C\) 的非负数解。

任取一个初始值 \(x_0\)，找函数图像上的点 \((x_0, x_0^2 - C)\)，沿该点做一条斜率为 \(f'(x_0) = 2x_0\) 切线，切线与 \(x\) 轴的交点的横坐标记为 \(x_1\)，\(x_1\) 比 \(x_0\) 更接近 \(\sqrt{C}\)。

切线方程为 \(y - (x_0^2 - C) = 2x_0(x - x_0)\)，令 y = 0 求得该切线与 \(x\) 轴交点的横坐标：

\[x_1 = \frac{C - x_0^2}{2x_0} + x_0 = \frac{1}{2}(x_0 + \frac{C}{x_0})\]

不断迭代，\(x_k\) 则不断地向 \(\sqrt{C}\) 靠近，那么何时可以结束？一般来说，可以判断相邻两次迭代的结果的差值是否小于一个极小的非负数 \(\epsilon\)，其中 \(\epsilon\) 一般可以取 \(10^{-6}\) 或 \(10^{-7}\)。

袖珍计算器

使用指数函数和对数函数代替平方根函数：

\[\sqrt x = x^{1/2} = (e^{\ln x})^{1/2} = e^{\frac{1}{2} \ln x}\]

注意：由于计算机无法存储浮点数的精确值，而指数函数和对数函数的参数和返回值均为浮点数，因此运算过程中会存在误差。例如当 \(x = 2147395600\) 时，\(e^{\frac{1}{2} \ln x}\) 的计算结果与正确值 \(4634046340\) 相差 \(10^{-11}\)，这样在对结果取整数部分时，会得到 \(4633946339\) 这个错误的结果。

因此在得到结果的整数部分 \(ans\) 后，我们应当找出 \(ans\) 与 \(ans+1\) 中哪一个是真正的答案。
