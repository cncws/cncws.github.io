Regular Expression Matching

Given an input string (s) and a pattern (p), implement regular expression matching with support for '.' and '*'.

The matching should cover the entire input string (not partial).

Note:

  • s could be empty and contains only lowercase letters a-z.

Recursion

Time complexity: \(O((m+n)2^{m+\frac{n}{2}})\), where m is the length of text and n is the length of pattern.

Space complexity: \(O((m+n)2^{m+\frac{n}{2}})\).

Dynamic Programming

dp[i][j] denotes wether text[i..m] matches pattern[j..n]. Indices start from 0.

Base cases:

  • dp[i][n] = False. when i = 0, 1, ..., m-1. Because text[i:] != '' but pattern[n:] == ''.

  • dp[m][n] = True. Because text[m:] == '' and pattern[n:] == ''.
