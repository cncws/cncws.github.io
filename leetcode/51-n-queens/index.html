N 皇后

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

上图为 8 皇后问题的一种解法。

每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。

  • 皇后彼此不能相互攻击，也就是说：任何两个皇后都不能处于同一条横行、纵行或斜线上。

回溯

判断对角线的方法：

  • i - j == r - c 时处于 左上-右下 斜对角线上，如 x 与 y。
  • i + j == r + c 时处于 坐下-右上 斜对角线上，如 x 与 z。

※ 集合 + 回溯

isValid() 方法：检查当前位置 (i,j) 与之前放置的皇后位置是否冲突。其时间复杂度与递归深度成正比。

本方案：每放置一枚皇后，将与当前皇后位置冲突的“位置”放入集合中，之后再检查 (i,j) 是否合法时，只需验证 (i,j) 是否位于冲突“位置”。其时间复杂度始终为 O(1)。

位运算 + 回溯

定义 columns, diagonal1, diagonal2 三个整数，且二进制表示中 1 代表不能放置皇后的位置，0 代表可以放置皇后的位置，最低位到最高位分别对应 0 到 n-1 列。

以 8 皇后问题为例，若不能放置皇后的位置为 3 和 6，则对应的二进制表示为 0100 1000。

columns, diagonal1, diagonal2 的计算规则：

  1. 初始时 columns = diagonal1 = diagonal2 = 0。

  2. 若在当前行的第 i 列放置了皇后，首先将 columns, diagonal1, diagonal2 的从低到高第 i 个二进制位置 1。

    例如在第一行的第 0 列，放置了皇后，则 columns = 0000 0001，diagonal1 = 0000 0001，diagonal2 = 0000 0001。

  3. 进入下一行之前，columns 保持不变。

    令 diagonal1 代表左下右上斜对角线上的位置，则 diagonal1 的第 i - 1 位应该置 1，即 diagonal1 应该向右移 1 位。

    令 diagonal1 代表左上右下斜对角线上的位置，同理，diagonal2 应该左移 1 位。

  4. 进入下一行时，通过按位或运算可以得出不能放置皇后的位置：columns | diagonal1 | diagonal2。

知道哪些位置不能放置皇后之后，我们希望遍历可以放置皇后的位置并放置皇后，这需要用到两个按位与运算的性质：

  • x & -x 可以只保留 x 的二进制表示中最低位的 1。
  • x & (x - 1) 可以将 x 的二进制表示中最低位的 1 置为 0。

这里需要用 1 代表可以放置皇后的位置，所以对 columns | diagonal1 | diagonal2 的结果取反，并且只保留低 n 位即可：(~(columns | diagonal1 | diagonal2)) & ((1 << n) - 1)。

怎么计算才能得到最低位的 1 所在位置呢？这里用的数 1 法：8 - 1 的结果为 7，7 的二进制表示中有 3 个 1，对应 3 号位。
