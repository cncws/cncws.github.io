组合

给定两个整数 n 和 k，返回 1 ... n 中所有可能的 k 个数的组合。

示例:

递归 + 回溯

固定第 i 个数，从 i+1 ... n 中选择 k-1 个数。

为了确保 i+1 ... n 中有至少 k-1 个数，需满足 n - (i+1) + 1 >= k-1，即 i <= n - k + 1。

※ 迭代枚举

从一个组合开始，依次枚举出所有组合，有两种枚举方法：

  1. 从后向前扫描当前组合，寻找能加一的位置 i，令 cur[i] += 1。为了不遗漏组合，cur[i+1] ... cur[k] 需要依序重置为 cur[i]+1 ... cur[k-1]+1。
  2. 从前向后扫描当前组合，寻找能加一的位置 i，令 cur[i] += 1。为了不遗漏组合，cur[0] ... cur[i-1] 需要依序重置为 1 ... i。
