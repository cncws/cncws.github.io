42. 接雨水

接雨水

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。感谢 Marcos 贡献此图。

示例:

暴力

单个柱子的积雨高度 等于 两侧最高的柱子中较低柱子的高度 减 当前柱子的高度。

动态规划

动态规划优化的地方：记录柱子的最大高度，避免重复查询。

left_max[i] 为索引小于等于 i 的最高柱子的高度；right_max[i] 为索引大于等于 i 的最高柱子的高度。

双指针

统计单个柱子的积雨高度：res += min(left_max[i], right_max[i]) - height[i]，因为同时用到 left_max 和 right_max，所以动态规划至少需要两次扫描才能得出结果。

更进一步，我们可以在一次扫描中得出结果。准备两个指针 left = 0, right = n - 1，同时维护两个变量 lmax = rmax = 0 记录两侧柱子的最大高度（等价于 lmax = max(height[:left + 1], rmax = max(height[right:n]）。left 向右移动，right 向左移动，且每次只移动较矮的柱子：

  • 当左侧柱子较矮（height[left] < height[right]）时：
    • lmax = max(lmax, height[left])（更新最大高度）
    • res = lmax - height[left]（累加积雨高度）
    • left += 1（向右移动 left）
  • 当右侧柱子较矮（height[left] >= height[right]）时：
    • rmax = max(rmax, height[right])
    • res += rmax - height[right]
    • right -= 1（向左移动 right）

当程序结束时，left = right 且指向最高的柱子。
