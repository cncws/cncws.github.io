53. 最大子序和

更新于 2020-10-29 分类于 编程 ， 力扣 阅读次数： Valine：

最大子序和

给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

示例:

进阶:

如果你已经实现复杂度为 O(n) 的解法，尝试使用更为精妙的分治法求解。

动态规划

令 dp[i] 为以 nums[i] 结尾的具有最大和的连续子数组。

则 dp[i + 1] = nums[i] + max(dp[i], 0)，求出完整的 dp 数组，取其最大值即可。

由于 dp[i + 1] 只依赖于 dp[i]，所以可使用一个变量代替数组，从而降低空间复杂度。

※ 分治算法

定义一个操作 get(a, l, r) 表示查询 a 序列 [l, r] 区间内的最大子段和，最终要求的答案就是 get(a, 0, len(a) - 1)。

分治：对于一个区间 [l, r]，取其中点 m = (l + r) // 2，对区间 [l, m] 和 [m + 1, r] 分治求解。当区间长度为 1 时，递归开始回升。

合并：如果通过 [l, m] 和 [m + 1, r] 的结果合并得到 [l, r] 的结果呢？需要维护哪些信息？

  1. 对于区间 [l, r] 维护四个信息：
    • lSum: [l, r] 区间中以 l 为左端点的最大子段和。
    • rSum: [l, r] 区间中以 r 为右端点的最大子段和。
    • mSum: [l, r] 区间的最大子段和。
    • iSum: [l, r] 的区间和。
  2. 设 [l, m] 的四个信息为 sl，[m + 1, r] 的四个信息为 sr，如何合并求出 [l, r] 的四个信息 s？
    • s.iSum = sl.iSum + sr.iSum
    • s.lSum = max(sl.lSum, sl.iSum + sr.lSum)
    • s.rSum = max(sr.rSum, sl.rSum + sr.iSum)
    • s.mSum = max(sl.mSum, sr.mSum, sl.rSum + sr.lSum)

iSum 区间和最简单，左右两个区间的区间和相加即可。

lSum 有两种情况：1. 右端点没有跨过 m； 2. 右端点跨过了 m。

rSum 的情况与 lSum 类似。

mSum 有三种情况：1. 最大子段完全位于左区间； 2. 完全位于右区间； 3. 左右区间分别有一部分。
