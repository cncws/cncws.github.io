map filter 函数

更新于 2020-08-04 分类于 编程 阅读次数： Valine：

map

map(function, iterable, ...)

将 iterable 中的元素依次传入 function，返回 function 的返回值。相当于

filter

filter(function, iterable)

将 iterable 中的元素依次传入 function，返回 function 的返回值为真时的元素。

当 function != None 时，相当于 (item for item in iterable if function(item))；

当 function == None 时，相当于 (item for item in iterable if item)。
