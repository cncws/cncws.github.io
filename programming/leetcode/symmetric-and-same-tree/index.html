Recursion

Define a function helper(t1, t2), then return true if t1 and t2 is symmetric. A tree root is symmetric only if root.left and root.right are symmetric.

Iteration

The helper(t1, t2) is actually a recursive traversal function, which visit two trees at the same time. The order to visit t1 is node -> left -> right (preorder), but the order to visit t2 is node -> right -> left (modified preorder).
