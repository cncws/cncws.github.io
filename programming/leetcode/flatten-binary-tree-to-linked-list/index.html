Recursion

Pre-order is not usable in recursion, because we modified pre.right before we visit it.

To avoid it, we can use order right -> left -> node, i.e., visit right, then left, then modify pointers (cur.right, cur.left = pre, None). Since we've visited those pointers before, we can modify them safely.
