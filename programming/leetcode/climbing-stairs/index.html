Climbing Stairs

更新于 2020-05-24 分类于 编程 Valine：

You are climbnig a stair case. It takes n steps to reach the top.

Each time you can either climb 1 or 2 steps. In how many distinct ways can you climb to the top?

Note: Given n will be a positive integer.

Dynamic Programming

Let dp[i] denotes the number of ways to reach on i-th step.

Base cases:

  • dp[1] = 1.
  • dp[2] = 2.

One can reach i-th step in one of the two ways:

  1. Taking a single step from (i - 1)-th step.
  2. Taking a step of 2 from (i - 2)-th step.

So, the total number of ways to reach i-th is equal to sum of ways of reaching (i - 1)-th step and ways of reach (i - 2)-th step. i.e. dp[n] = dp[n - 1] + dp[n - 2].

Fibonacci Number

In the above approach we have used dp array where dp[i] = dp[i - 1] + dp[i - 2]. It can be easily analysed that dp[i] is nothing but i-th fibonacci number.

Binets Method

This is an interesting solution which uses matrix multiplication to obtain the n-th Fibonacci Number. The matrix takes the following form: \(\begin{bmatrix}1&1\\1&0\end{bmatrix}\).

Let's say \(Q^n=\begin{bmatrix}F_n&F_{n-1}\\F_{n-1}&F_{n-2}\end{bmatrix}\), the n-th Fibonacci Number is given by \(Q^n[0][0]\).

In particular, we denote \(Q^1=\begin{bmatrix}1&1\\1&0\end{bmatrix}\), where \(F_1=Q^1[0][0]=1\).

We can prove this method using Mathematical Induction.

\(Q^2=Q^1\begin{bmatrix}1&1\\1&0\end{bmatrix}=\begin{bmatrix}2&1\\1&1\end{bmatrix} \quad F_2=Q^2[0][0]=2\) holds.

\(Q^3=Q^2\begin{bmatrix}1&1\\1&0\end{bmatrix}=\begin{bmatrix}3&2\\2&1\end{bmatrix} \quad F_3=Q^3[0][0]=3\) holds.

Thus, the base case holds.

Assume that this method holds for finding the n-th Fibonacci Number, i.e. \(F_n = Q^n[0][0]\), where, \(Q^n=\begin{bmatrix}F_n&F_{n-1}\\F_{n-1}&F_{n-2}\end{bmatrix}\).

Now, we need to prove that with the above two conditions holding true, the method is valid for finding the (n + 1)-th Fibonacci Number, i.e. \(F_{n+1} = Q^{n+1}[0][0]\).

Proof: \(Q^{n+1}=\begin{bmatrix}F_n&F_{n-1}\\F_{n-1}&F_{n-2}\end{bmatrix}\begin{bmatrix}1&1\\1&0\end{bmatrix}=\begin{bmatrix}F_n+F_{n-1}&F_n\\F_{n-1}+F_{n-2}&F_{n-1}\end{bmatrix} = \begin{bmatrix}F_{n+1}&F_n\\F_n&F_{n-1}\end{bmatrix}\).

Thus, \(F_{n+1} = Q^{n+1}[0][0]\). This completes the proof of this method.

Proof of time complexity:

Suppose, \(n\) is the power of 2. To calculate \(Q^n\) matrix, we should calculate \(Q^{n/2}\) matrix and multiply it by itself. To calculate \(Q^{n/2}\) we should have to do the same with \(Q^{n/4}\) and so on. Meaning that: \(Q^{n}=Q^{n/2}\cdot Q^{n/2}=\dots=\prod_1^n Q^1\).

e.g. \(Q^8=(Q^4)^2=((Q^2)^2)^2=(((Q^1)^2)^2)^2\).

Obviously, the tree height is \(\log_2 n\).

In case, the number \(n\) is not a power of 2, we can break it in terms of powers of 2 using its binary representation: \(n=\sum_{p\in P}2^p\), where \(P\subset \N\).

Thus, we can obtain the final result using: \(Q^n=\prod_{p\in P}Q^{2^p}\).

e.g. \(Q^7=Q^4\cdot Q^2 \cdot Q^1\).

This is the method we've used in implementation. Again, the complexity remains \(O(\log_2 n)\) as we have limited the number of multiplications to \(O(\log_2 n)\).

Fibonacci Formula

We can find n-th fibonacci number using this formula: \(F_n=1/\sqrt 5\Big[\Big(\frac{1+\sqrt 5}{2}\Big)^n-\Big(\frac{1-\sqrt 5}{2}\Big)^n\Big]\).

For the given problem, the Fibonacci sequence is defined by \(F_0=1, F_1=1, F_2=2, F_{n+2}=F_{n+1}+F_n\).

A standard method of trying to sovle such recursion formulas is assume \(F_n\) of the form \(F_n=a^n\).

Then, of cource, \(F_{n+1}=a^{n+1}\) and \(F_{n+2}=a^{n+2}\) so the equation becomes \(a^{n+2}=a^{n+1}+a^n\). If we divide the entire equation by \(a^n\) we arrive at \(a^2=a+1\).

Solving this by the quadratic formula, we get: \(a=1/\sqrt 5\Big(\frac{1\pm\sqrt 5}{2}\Big)\).

The general solution, thus takes the form: \(F_n=A\Big(\frac{1+\sqrt 5}{2}\Big)^n+B\Big(\frac{1-\sqrt 5}{2}\Big)^n\).

For \(n=0\), we get \(A+B=1\). For \(n=1\), we get \(A\Big(\frac{1+\sqrt 5}{2}\Big)+B\Big(\frac{1-\sqrt 5}{2}\Big)=1\). Solving the above equations, we get \(A=\Big(\frac{1+\sqrt 5}{2\sqrt 5}\Big), B=\Big(\frac{1-\sqrt 5}{2\sqrt 5}\Big)\).

Putting these values of \(A\) and \(B\) in the above general solution equation, we get:

\(F_n=1/\sqrt 5\Big[\Big(\frac{1+\sqrt 5}{2}\Big)^n-\Big(\frac{1-\sqrt 5}{2}\Big)^n\Big]\)
