数字圆环

更新于 2020-08-09 分类于 编程 阅读次数： Valine：

一个长度为 n 的数字数组 a。问是否能用这 n 个数字构成一个环（首尾连接），使得环中的每一个数字都小于它相邻的两个数字的和（每个数字都必须使用并且每个数字只能使用一次）。

输入描述:

第一行包含一个整数 t(1 <= t <= 10)，表示测试用例的组数。

每个测试用例输入如下：

第一行一个整数 n，表示数字的个数；

第二行 n 个整数，每两个整数之间用一个空格分隔。

输入保证：3 <= n <= 1e5；1 <= a[i] <= 1e9

输出描述:

输出应该包含 t 行，对于每组用例，若能输出 "YES"，否则输出 "NO"。

解

什么样的数组可以组成符合题目要求的环？

首先按照升序排序的三个数字，a[i-1], a[i], a[i+1]，a[i] 一定小于 a[i-1] + a[i+1]。唯一可能违反要求的位置在于最后一个元素 a[n]。

又如：1 3 7 8 就无法组成符合要求的圆环。

程序需要做的是：判断两个元素后能否在不违反要求的前提下交换，即，在交换 a[n-1], a[n] 之前，满足 a[n] > a[n-1] + a[n-2]。
