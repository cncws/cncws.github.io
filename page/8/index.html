Dynamic Programming

Let dp_max[i] be the maximum product of subarray and dp_min[i] be the minimum product of subarray that ends with nums[i]. The maximum value of array dp_max is the answer.

Base case: dp_max[0] = dp_min[0] = nums[0].

Induction rules:

  • If nums[i] >= 0, dp_max[i] = max(dp_max[i - 1] * nums[i], nums[i]), dp_min[i] = min(dp_min[i - 1] * nums[i], nums[i]).
  • If nums[i] < 0, dp_max[i] = max(dp_min[i - 1] * nums[i], nums[i]), dp_min[i] = min(dp_max[i - 1] * nums[i], nums[i]).

Triangle

Given a triangle, find the minimum path sum from top to bottom. Each step you may move to adjacent numbers on the row below.

For example, given the follwing triangle

The minimum path sum from top to bottom is 11 (i.e., 2 + 3 + 5 + 1 = 11).

Dynamic Programming

Let dp[i][j] denotes the sum of minimum path that end at triangle[i][j].

Base case: dp[m - 1][j] = triangle[m - 1][j]. Where m is the length of triangle.

Induction rule: dp[i][j] = triangle[i][j] + min(dp[i + 1][j], dp[i + 1][j + 1]).

Example 2:

Brute Force

The most basic idea is to find every string possible by all interleavings of the two given strings s1 and s2. m is the length of s1, n is the length of s2.

If s3[0 .. i + j - 1] is formed by the interleaving of s1[0 .. i - 1] and s2[0 .. j - 1],

  • If s3[i + j] == s1[i], then check if s3[i + j + 1 .. m + n - 1] can be obtained by s1[i + 1 .. m - 1] and s2[j .. n - 1].
  • If s3[i + j] == s2[j], then check if s3[i + j + 1 .. m + n - 1] can be obtained by s1[i .. m - 1] and s2[j + 1 .. n - 1].
  • Else, return false.
