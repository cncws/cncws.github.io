最小路径和

给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

示例:

动态规划

令 dp[i][j] 表示从 (i, j) 到右下角的最小路径和。

最后一行只能向右走，最后一列只能向下走，因此：

  • 最后一行有：dp[m - 1][j] = grid[m - 1][j] + dp[m - 1][j + 1]。
  • 最后一列有：dp[i][n - 1] = grid[i][n - 1] + dp[i + 1][n - 1]。

转移方程：dp[i][j] = grid[i][j] + min(dp[i + 1][j], dp[i][j + 1])。

网格中的障碍物和空位置分别用 1 和 0 来表示。

说明：m 和 n 的值均不超过 100。

示例 1:

动态规划

示例 2:

提示：

  • 1 <= m, n <= 100
  • 题目数据保证答案小于等于 2 * 10 ^ 9

动态规划

令 dp[i][j] 表示从 (i, j) 到达右下角 (m - 1, n - 1) 的不同路径数，dp[0][0] 就是问题的解。

转移方程：(i, j) 的下一步可以是 (i + 1, j) 或 (i, j + 1)，因此，dp[i][j] = dp[i + 1][j] + dp[i][j + 1]。

特殊地：

  1. 处于最后一行时：dp[m - 1][j] = 1。
  2. 处于最后一列时：dp[i][n - 1] = 1。

由于 dp[i][j] 的结果只依赖于同列的下一行和同行的下一列，可以使用一维数组代替二维数组（该技巧称为滚动数组），从而降低空间复杂度。
