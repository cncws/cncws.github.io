奖学金

有 n 门课，每门课由平时成绩和考试成绩组成，满分为 r。为了拿到奖学金，平均成绩至少为 avg。

现在知道每门课的平时成绩为 a[i]，若想让这门课的考试成绩多拿一分的话，要花 b[i] 的时间复习。复习得再多也不会拿到超过满分的分数。

为了拿到奖学金，至少要花多少时间复习。

输入描述：

第一行三个整数 n,r,avg (1 <= n <= 1e5, 1 <= r <= 1e9, 1 <= avg <= 1e6)

接下来 n 行，每行两个整数 a[i] 和 b[i]，1 <= a[i], b[i] <= 1e6

输出描述：

一行输出答案

示例1：

贪心算法

优先复习“容易”的学科。

分苹果

有 n 堆，每堆分别有 a[i] 个苹果。现在你要在它们之间转移苹果，使得每一堆中的苹果数都相同。每一次，你只能从一堆中拿走恰好两个苹果到另一堆中，问最少需要移动多少次可以平分苹果，如果方案不存在输出 -1。

输入描述：

第一行包含一个整数 n (1 <= n <= 100)

接下来的一行包含 n 个整数 a[i] (1 <= a[i] <= 100)

输出描述：

输出一行表示最少需要移动多少次可以平分苹果，如果方案不存在则输出 -1。

示例 1：

贪心算法

若苹果最终能被均分，拿出的次数与放入的次数一定是相等的，只需计数其中一个就行。

只需要依次检查每一堆，缺少或多余的苹果是否是 2 的整数倍，若不是，则问题无解。
