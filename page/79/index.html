恐龙蛋

更新于 2020-08-04 分类于 编程

给定恐龙蛋的个数 n 和大小数组。恐龙蛋的大小各不相同，按从大到小依次排列，第 i 个恐龙蛋每天增大 i，求最少几天后会出现两个同样大小的恐龙蛋。

解

相邻的恐龙蛋增长速度相差为 1。因此，如果初始时恐龙蛋大小的差异数组 d = [d1, d2, ..., d{n-1}]，那么每过 1 天，数组 d 中每个元素都会减 1。当 d 中首次出现 0 元素时，就是首次出现两个相同大小恐龙蛋的时候。（首次出现大小相同的两个蛋一定相邻）
