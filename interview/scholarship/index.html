奖学金

有 n 门课，每门课由平时成绩和考试成绩组成，满分为 r。为了拿到奖学金，平均成绩至少为 avg。

现在知道每门课的平时成绩为 a[i]，若想让这门课的考试成绩多拿一分的话，要花 b[i] 的时间复习。复习得再多也不会拿到超过满分的分数。

为了拿到奖学金，至少要花多少时间复习。

输入描述：

第一行三个整数 n,r,avg (1 <= n <= 1e5, 1 <= r <= 1e9, 1 <= avg <= 1e6)

接下来 n 行，每行两个整数 a[i] 和 b[i]，1 <= a[i], b[i] <= 1e6

输出描述：

一行输出答案
