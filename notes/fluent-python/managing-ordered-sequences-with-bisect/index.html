用bisect来管理已排序的序列

更新于 2020-01-12 分类于 笔记 Valine：

bisect

MethodDoc
bisect.bisect_left(a, x, lo=0, hi=len(a))Return the leftmost index where to insert item x in list a, assuming a is sorted.
bisect.bisect_right(a, x, lo=0, hi=len(a))Return the rightmost index where to insert item x in list a, assuming a is sorted.
bisect.bisect(a, x, lo=0, hi=len(a))Equal to bisect_right.
bisect.insort_left(a, x, lo=0, hi=len(a))Insert item x in a and keep it sorted assuming a is sorted. If equal keys are found, add to the left.
bisect.insort_right(a, x, lo=0, hi=len(a))Insert item x in a and keep it sorted assuming a is sorted. If equal keys are found, add to the right.
bisect.insort(a, x, lo=0, hi=len(a))Equal to insort_right.

SortedCollection

Python的高产贡献者Raymond Hettinger写了一个排序集合模块SortedCollection，模块里集成了bisect功能，但是比独立的bisect更易用。
